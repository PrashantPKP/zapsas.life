## 🎯 About ZAPSAS

**ZAPSAS** is a comprehensive digital platform that brings together productivity tools, browser-based games, innovative web applications, and educational articles - all in one place. Built with modern web technologies, ZAPSAS provides a seamless, free, and accessible experience across all devices.


### 🌟 What We Offer

- **9+ Free Online Tools** - Productivity utilities for everyday tasks
- **6+ Browser Games** - Classic and modern games, no downloads required
- **6+ Web Projects** - Innovative applications solving real-world problems
- **174+ Articles** - Educational content covering 11+ technology categories
- **100% Free** - No subscriptions, no hidden fees
- **Privacy-First** - Secure, GDPR-compliant platform

## 🔧 Technology Stack

### Frontend

- **Framework:** React 18.2.0
- **Routing:** React Router DOM (for subdomains)
- **Styling:** CSS3, Bootstrap 5.3.2
- **Icons:** Font Awesome
- **SEO:** React Helmet Async
- **Security:** DOMPurify 3.3.0

### Backend (Projects & Tools)

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB (for projects requiring persistence)
- **Authentication:** JWT (where applicable)
- **API Integration:** Various third-party APIs

### Development Tools

- **Build Tool:** React Scripts 5.0.1
- **Package Manager:** npm
- **Version Control:** Git
- **Code Quality:** ESLint
- **Deployment:** cPanel (Production)

### Security Stack

- **Sanitization:** DOMPurify
- **HTTPS:** SSL/TLS encryption
- **CSP:** Content Security Policy
- **Input Validation:** Custom security utilities
- **Rate Limiting:** Custom rate limiter implementation
